# <img width="30" height="30" alt="pngwing com" src="https://github.com/user-attachments/assets/388800e7-1c5a-493f-b0e6-85e33474d0cc" /> AirBNB A-B test report

A report on AirBNB open source data made using MySQL and PowerBi. The purpose of the report is to analyse the overall performance of the AirBNB company, and how it manages and maintains customer relations in two different scenarios.

## 🪛 Technologies

- `MySQL`
- `PowerBI`
- `MS Word`
- `Sublime Text`

## 👩🏽‍🍳 The Process

- I collected the data from the AirBNB website.
- Ran the SQL querries.
- Connected the workbench with PowerBI.
- Created a dashboard using PowerBI
- Performed analysis on the insights/visualisations.
- Created a report of recommendations.

## 📋 Key Findings

- **Higher Reviews:** Customer reviews are up by 19.6% when instant bookings are available.
- **Price Difference:**  When instant bookings were not available, the average prices were 1.6% higher in case of “No Instant Booking”.
- **Lower Ratings:** The average ratings of the business were down by 3.7% when instant bookings were available.

## ⚙️ Running the Project

To run the project in your local environment:

1. Download the [airbnb_open_data.csv](https://drive.google.com/file/d/1nJSAdjEH5knhpvTepNhUcpgGSkUCTmGn/view?usp=drive_link) file and the [airbnb_workbench.sql] file into your system.
2. Run the .sql file using any sql workbench.
3. Open PowerBI and connect the dataset to PowerBI. (You can watch this [tutorial video](https://youtu.be/FprIrHOHvPo) for setting-up)
4. Do your own analysis through visualisations and dashboards.
