A report on AirBNB open source data made using MySQL and PowerBi. The purpose of the report is to analyse the overall performance of the AirBNB company, and how it manages and maintains customer relations in two different scenarios.

## 🪛 Technologies

- `MySQL`
- `PowerBI`
- `MS Word`
- `Sublime Text`

## 👩🏽‍🍳 The Process

- I collected the data from the AirBNB website.
- Ran the SQL querries.
- Connected the workbench with PowerBI.
- Created a dashboard using PowerBI
- Performed analysis on the insights/visualisations.
- Created a report of recommendations.
